# プログラミング言語

## おそらく学ぶべき言語
### C
全ての基本。仕様自体はシンプル。マイコンや低レベルのソフトウェアを記述するときに使う。高速だが安全でもモダンでもない。

### C++
C言語にオブジェクト指向を追加した言語。基本構文はCと同じだが、実際のところCpp11以降のモダン構文を使うことが多いため、結構違う。ROS2で一番メインで使われる言語。これがないとロボットが動かない。高速、言語仕様はでかい。

### Python 
ROS2でのもう一つの選択肢。実行速度が遅いが書きやすく、情報もライブラリも豊富。制御に使うことはさけるべきだが、テスト用のコードや画像認識周りでは非常に便利なので結構使う。

## 使えると役に立つかもしれない言語


### JavaScript/TypeScript
Webで覇権取った言語。UI作成に便利なのと、汎用性はかなり高いのでやって損はないかなという言語。CppやRustほど速くないが、JIT芸でかなり高速化されている。

### Rust
速いしモダンで将来性もありそう。ただ、ビルド遅い、構文が特殊、多言語ほどライブラリが揃ってない。あと環境構築もちょっと面倒。

## 他の有名言語について一言ずつ

### Java
安定。冗長。Web等のシステムで企業がよく使う。

### PHP
基本Webバックエンド用。Laravelで生き残ってる老人。

### Ruby
珍しい日本初の言語。webで見るが人気下降中な気もしてしまう。

### Kotlin
Android開発ならコイツ。Javaの後継。

### Swift
iOS開発で使う。誰も覚えていないObjectiveCの後継。

### Go
DevOps系ツールでよく使われてる。googleが開発したシンプルな言語。

### Lisp
神の言語。関数型の走りみたいなやつ。

### C\#
Javaに.Net付けたMicrosoftの巨人。Unityで使う。
